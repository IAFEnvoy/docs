# Keybindings

### Origins keybindings

| Identifier | In-Game Name | Default |
|------------|--------------|---------|
| `key.origins.primary_active` | Active Power (Primary) | `G` |
| `key.origins.secondary_active` | Active Power (Secondary) | *Unbound* |
| `key.origins.view_origin` | View Origin | `O` |
