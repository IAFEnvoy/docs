- [Avian](./avian) (`origins:avian`)
- [Arachnid](./arachnid) (`origins:arachnid`)
- [Elytrian](./elytrian) (`origins:elytrian`)
- [Shulk](./shulk) (`origins:shulk`)
- [Feline](./feline) (`origins:feline`)
- [Enderian](./enderian) (`origins:enderian`)
- [Merling](./merling) (`origins:merling`)
- [Blazeborn](./blazeborn) (`origins:blazeborn`)
- [Phantom](./phantom) (`origins:phantom`)
