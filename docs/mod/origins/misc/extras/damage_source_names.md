# Damage Source Names

Damage Source Names are strings used for classifying what type of damage is taken/given, and to determine the death message for the damage upon being killed.

### List of vanilla damage source names

| Name | Description |
|------|-------------|
| `anvil` | Being hit by a falling Anvil. |
| `anvil.player` | Being hit by a falling Anvil while fighting a player/mob. |
| `arrow` | Being shot by a player/mob. |
| `arrow.item` | Being shot by a player/mob using a renamed Bow or Crossbow. |
| `badRespawnPoint` | Being hit by an explosion caused by Beds from attempting to use it in The End, The Nether or custom dimensions. |
| `cactus` | Making contact with a Cactus. |
| `cactus.player` | Making contact with a Cactus while fighting a player/mob. |
| `cramming` | Being in the same spot as many other entities. |
| `cramming.player` | Being in the same spot as many other entities while fighting a player/mob. |
| `dragonBreath` | Being damaged by the Dragon's breath. (Dragon breath deals `magic` damage in vanilla.) |
| `dragonBreath.player` | Being damaged by the Dragon's breath while fighting a player/mob. |
| `drown` | Drowning. |
| `drown.player` | Drowning while fighting a player/mob. |
| `dryout` | Drying out. **Unused in vanilla.** |
| `dryout.player` | Drying out while fighting a player/mob. **Unused in vanilla.** |
| `even_more_magic` | If the death message is too large to send. **Unused in vanilla.** |
| `explosion` | Being hit by a regular explosion. |
| `explosion.player` | Being hit by an explosion from a mob or TNT lit by a player/mob. |
| `fall` | Falling from a high place. |
| `fall.player` | Falling from a high place while fighting a player/mob. |
| `fallingBlock` | Being hit by a falling block that's not an Anvil. |
| `fallingBlock.player` | Being hit by a falling block while fighting a player/mob. |
| `fallingStalactite` | Being hit by a falling Stalactite block. |
| `fallingStalactite.player` | Being hit by a falling Stalactite while fighting a player/mob. |
| `fireball` | Being hit by a fireball shot by a player/mob. |
| `fireball.item` | Being hit by a fireball shot by a player/mob holding a renamed item. |
| `fireworks` | Being hit by a firework explosion. |
| `fireworks.player` | Being hit by a firework explosion while fighting a player/mob. |
| `fireworks.item` | Being hit by a firework explosion shot by a player/mob using a renamed Crossbow. |
| `flyIntoWall` | Flying into a wall at high speed using an Elytra. |
| `flyIntoWall.player` | Flying into a wall at high speed using an Elytra while fighting a player/mob. |
| `freeze` | Being in Powder Snow for too long. |
| `freeze.player` | Being in Powder Snow for too long while fighting a player/mob. |
| `generic` | Being damaged by a damage source without a specific type. |
| `generic.player` | Being damaged by a damage source without a specific type while fighting a player/mob. |
| `hotFloor` | Standing on Magma. |
| `hotFloor.player` | Standing on Magma while fighting a player/mob. |
| `indirectMagic` | Being damaged by indirect magic damage (e.g. thrown Splash Potions). |
| `indirectMagic.item` | Being damaged by indirect magic damage while fighting a player/mob with a renamed item. |
| `inFire` | Standing inside Fire. |
| `inFire.player` | Standing inside Fire while fighting a player/mob. |
| `inWall` | Being inside a solid block. |
| `inWall.player` | Being inside a solid block while fighting a player/mob. |
| `lava` | Being in Lava. |
| `lava.player` | Being in Lava while fighting a player/mob. |
| `lightningBolt` | Being struck by a Lightning Bolt. |
| `lightningBolt.player` | Being struck by a Lightning Bolt while fighting a player/mob. |
| `magic` | Being damaged by magic damage (e.g. Conduits, Instant Damage, Poison). |
| `magic.player` | Being damaged by magic damage while fighting a player/mob. |
| `mob` | Being hit by a mob. |
| `mob.item` | Being hit by a mob holding a renamed item. |
| `onFire` | Burning. |
| `onFire.player` | Burning while fighting a player/mob. |
| `outOfWorld` | Falling into the void. |
| `outOfWorld.player` | Falling into the void while fighting a player/mob. |
| `player` | Being hit by a player. |
| `player.item` | Being hit by a player holding a renamed item. |
| `sonic_boom` | Being hit by a Warden's Sonic Boom. |
| `sonic_boom.player` | Being hit by a Warden's Sonic Boom while fighting a player/mob. |
| `sonic_boom.item` | Being hit by a Warden's Sonic Boom while fighting a player/mob using a renamed item. |
| `stalagmite` | Falling onto a Stalagmite. |
| `stalagmite.player` | Falling onto a Stalagmite while fighting a player/mob. |
| `starve` | Being damaged by starvation. |
| `starve.player` | Being damaged by starvation while fighting a player/mob. |
| `sting` | Being stung by a bee. |
| `sting.player` | Being stung by a player/mob. **Unused in vanilla.** |
| `sting.item` | Being stung by a player/mob holding a renamed item. **Unused in vanilla.** |
| `sweetBerryBush` | Walking through a Sweet Berry Bush. |
| `sweetBerryBush.player` | Walking through a Sweet Berry Bush while fighting a player/mob. |
| `thorns` | Hitting a player/mob wearing Thorns armor (or Guardians/Elder Guardians). |
| `thorns.item` | Hitting a player/mob wearing Thorns armor while holding a renamed item. |
| `thrown.item` | Being hit by a projectile thrown by a player/mob. **Unused in vanilla.** |
| `trident` | Being hit by a Trident thrown by a player/mob. |
| `trident.item` | Being hit by a Trident thrown by a player/mob holding a renamed item. |
| `wither` | Being damaged by the Wither status effect. |
| `wither.player` | Being damaged by the Wither status effect while fighting a player/mob. |
| `witherSkull` | Being hit by a Wither Skull shot by a player/mob. |

### List of Origins damage source names

| Name | Description |
|------|-------------|
| `genericDamageOverTime` | Being damaged by a power that uses the `origins:damage_over_time` power type with no specified damage source name. |
| `genericDamageOverTime.player` | Being damaged by a `damage_over_time` power type while fighting a player/mob. |
| `hurt_by_water` | Being damaged by the `origins:water_vulnerability` power. |
| `hurt_by_water.player` | Being damaged by the `origins:water_vulnerability` power while fighting a player/mob. |
| `no_water_for_gills` | Losing air with the `origins:water_breathing` power. |
| `no_water_for_gills.player` | Losing air with the `origins:water_breathing` power while fighting a player/mob. |
