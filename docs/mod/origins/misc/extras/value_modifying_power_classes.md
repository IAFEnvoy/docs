# Value Modifying Power Classes

Value Modifying Power Classes are Java classes that extend the `ValueModifyingPower` super-class. These are used in the `modify` field of entity condition/action power types.

### List of Value Modifying Power Classes

|------------|--------------|

| `ModifyBreakSpeedPower` | `modify_break_speed` |
