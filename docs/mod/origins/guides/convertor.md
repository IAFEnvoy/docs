# Porting: Fast convertor (Beta)

:::caution
- This converter can only convert from 1.21.1 Fabric, if your datapack was made for 1.20.4 or earlier, you will need to manually fix version-specific changes (such as `ItemStack`) after conversion.

- The converter is in early beta and may not cover all edge cases. Review logs and converted files carefully and refer to the manual porting guide below for any discrepancies.
:::

Report any bugs or inaccuracies you find in the converter tool.
